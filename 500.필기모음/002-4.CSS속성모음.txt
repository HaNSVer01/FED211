[ 글자관련 ]
1. color : 글자색
2. font-family : 글자체
  
3. font-weight : - nomal | bold | border | 100~900 -> 글자체 지원에 따라 다름
글자스타일(이텔릭)
4. font-style
letter-spacing: /* 글자간격 */
word-spacing: /* 단어간격 */
text-align: 박스에서의 글자(인라인요소) 정렬
    left(기본값) | right | center | justify(양쪽정렬)- 글많은 문단에서 주로 사용

[ 박스모델관련 ]
1. width: 가로크기
2. height: 세로크기
3. padding: 안쪽여백
4. margin: 바깥여백
5. border: 경계선
    -> border: 두께 종류 색상(border-color);(두종색)
    - 두께 :(border-weght) (px, em)
    - 종류 :(border-style) solid(선형) , dotted(점선형), dashed

    -> 방향별 보더: 
    border-top,borderin-left. border-right, border-bottom

마진, 패딩 공통 셋팅!
margin: 값; -> 4방향
margin: 값 값; -> 위아래 양쪽
margin: 값 값 값; -> 위 양쪽 아래
margin: 값 값 값 값; -> 위 오른 아래 왼(시계방향)

paddig: 값; -> 4방향
paddig: 값 값; -> 위아래 양쪽
paddig: 값 값 값; -> 위 양쪽 아래
paddig: 값 값 값 값; -> 위 오른 아래 왼(시계방향)

-> 각 방향별 속성
margin-top, ,margin-left. margin-right, margin-bottom
paddig-top, ,paddig-leftpaddigin-right, paddig-bottom
