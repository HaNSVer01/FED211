[문자 데이터 관련 태그]
1. h1~h6 : 제목
2. p : 단락(내용구분)
    --> 블록태그임
    (주의 : 다른블록요소 내부에 못씀!)
3. a : 링크 (Anchor:닻)
    - href 속성: 이동할 곳
    - target 속성: 창옵션(_blank새창)
        (기본값은 _self 현재창)
4. br : 줄바꿈 (break 줄을 깸)
5. hr : 가로구분선 (horizontal 수평선)            
6. <b> - 굵은 텍스트
<strong> - 중요한 텍스트
<i> - 기울임꼴 텍스트
<em> - 강조된 텍스트
<mark> - 표시된 텍스트
<small> - 더 작은 텍스트
<del> - 삭제된 텍스트
<ins> - 삽입된 텍스트
<sub> - 아래 첨자 텍스트
<sup> - 위 첨자 텍스트
